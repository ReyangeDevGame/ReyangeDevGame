# Projet Antigravity : Simulateur d'Entretien IA

Bienvenue dans votre projet de session !

## 🏗 Structure du Projet

*   `src/` : C'est ici que votre IA Développeur (`[CODER]`) va placer le code source.
*   `docs/` : C'est le cerveau du projet.
    *   `specs/` : Où votre IA Architecte (`[ARCHITECT]`) va écrire les plans.
    *   `reports/` : Où votre IA Testeur (`[QA]`) va écrire les plans de test.
*   `prompt_book.md` : Votre "Grimoire" contenant les sorts pour activer vos IAs.

## 🚀 Mission 1 : Initialisation

1.  Ouvrez votre Chatbot (ChatGPT, Gemini...).
2.  Ouvrez le fichier `prompt_book.md`.
3.  Cherchez le prompt **BUSINESS ANALYST** et copiez-le dans le chat.
4.  Laissez-vous guider !

## 🛠 Installation Technique (Pour plus tard)

```bash
# Créer l'environnement virtuel
python -m venv .venv

# Activer
# Windows :
.venv\Scripts\activate
# Mac/Linux :
source .venv/bin/activate

# Installer les dépendances (quand vous en aurez)
pip install -r requirements.txt
```
